# プロジェクト定義: スライドテンプレート生成

## 概要
PowerPointテンプレート「JMDC2022_16対9(標準)_v1.1.pptx」をベースに、AIが生成しやすいスライドテンプレートを作成するプロジェクトです。

## 目標
1. **再現性**: 元のPPTXのデザイン（フォント、色、レイアウト）を可能な限り忠実に再現する。
2. **AI親和性**: LLMがコンテンツを充填しやすいMarkdown形式（Marp等）を採用する。
3. **構成**:
    - 表紙
    - 目次
    - 中扉（セクション区切り）
    - 本文（テキスト + 箇条書き）
    - 本文（2カラム）
    - グラフ・図表用レイアウト
    - 裏表紙

## 参照ファイル
- `JMDC2022_16対9(標準)_v1.1.pptx`: マスターテンプレート

## 技術スタック
- **Format**: Markdown (Marp)
- **Styling**: CSS
- **Output**: HTML / PDF / PPTX (via Marp)
