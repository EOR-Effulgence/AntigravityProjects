# 動画制作ワークフロー: ポンコツAIニュース

**使用ツール**:
*   **動画編集**: Adobe Premiere Pro
*   **音声合成**: VOICEVOX
*   **台本**: `script_vol1.md`

## 1. 事前準備 (素材配置)
プロジェクトフォルダ (`ponkotsu_ai_news`) に以下の素材があるか確認します。
*   `studio_background.png` (背景)
*   `character.png` (※キャスター画像。まだない場合は仮のものか、後で生成)
*   `material_samurai_sushi.png` (ネタ画像1)
*   `material_keyboard_sushi.png` (ネタ画像2)

## 2. 音声生成 (VOICEVOX)
1.  **VOICEVOXを起動**。
2.  **キャラクター設定**:
    *   **レイ (キャスター)**: 「九州そら」または「四国めたん」を選択。
    *   **G-GPT (AI)**: 「ずんだもん」や「ロボットボイスっぽいキャラ」を選択。
3.  **台本の入力**:
    *   `script_vol1.md` のセリフを一行ずつコピー＆ペースト。
    *   イントネーションを調整。
4.  **書き出し**:
    *   「音声書き出し」を実行。
    *   ファイル名は `01_news.wav`, `02_kensho.wav` のように連番にすると編集しやすいです。

## 3. Premiere Pro 編集手順

### (1) シーケンス設定 (縦動画)
*   **新規シーケンス**:
    *   フレームサイズ: **1080 x 1920** (9:16)
    *   フレームレート: **30fps** または **60fps**

### (2) タイムライン構築
トラック（レイヤー）を以下のように構成すると管理しやすいです。

| トラック | 配置するもの | 備考 |
| :--- | :--- | :--- |
| **V4** | テロップ (キャプション) | 自動文字起こし or エッセンシャルグラフィックス |
| **V3** | ネタ画像 (ワイプ/インサート) | サムライ寿司などの画像。適宜アニメーションさせる |
| **V2** | キャスター立ち絵 (レイ) | 口パクレイヤーがあれば重ねる |
| **V1** | スタジオ背景画像 | `studio_background.png` |
| **A1** | 音声 (VOICEVOX) | 生成したwavファイルを配置 |
| **A2** | BGM / SE | ニュースっぽいOP、ツッコミ効果音など |

### (3) 編集のコツ
*   **テンポ優先**: 音声の「間」は徹底的にカット（ジェットカット）。
*   **動きをつける**: 静止画の背景やキャラでも、「スケール」や「位置」をキーフレームで少し動かすだけでリッチに見えます。
*   **ツッコミ演出**: `G-GPT` のボケ発言の時は、BGMを止めたり、「？」マークを出したりすると効果的。

## 4. 書き出し
*   **形式**: H.264
*   **プリセット**: YouTube 1080p フルHD (ただし縦横比は維持)
*   **ファイル名**: `ponkotsu_news_vol1.mp4`
