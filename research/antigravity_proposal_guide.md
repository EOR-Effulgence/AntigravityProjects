# Google Antigravityを活用した提案書作成（PowerPoint編）

このドキュメントでは、AIエージェントを活用して**Markdownで原稿を作成し、PowerPoint形式の提案書として出力する**ためのワークフローを解説します。

## 1. コア戦略: "Markdown to PowerPoint"

PowerPointを直接AIにいじらせるのではなく、**「Markdownで内容と構成を完璧にし、変換ツールでスライド化する」** アプローチが最も効率的です。

*   **使用ツール:** [Marp (Markdown Presentation Ecosystem)](https://marp.app/)
    *   VS Code拡張機能「Marp for VS Code」を入れるだけで、MarkdownをPowerPoint (.pptx) に変換できます。
    *   AntigravityはMarp記法（スライド区切りやテーマ設定）を理解しています。

## 2. ワークフロー（PowerPoint出力版）

### シナリオA: 公開情報を元に新規作成
1.  **リサーチ:** 「〇〇業界の最新トレンドを箇条書きでまとめて」
2.  **構成案:** 「新規事業提案のスライド構成（全10枚）を考えて。Marpで出力することを想定して、1枚ごとのタイトルと要点を決めて」
3.  **スライド執筆:** 「各スライドの具体的な内容を書いて。図解が必要な箇所はMermaid記法を使って記述して」

### シナリオB: 既存提案書のリバイス／要約（推奨テスト）
手元にある長文のドキュメントや、古い提案書を今風のパワポにするパターンです。

1.  **読み込み:** 既存のテキストや提案書の内容をAntigravityに渡す（コンテキストとして共有）。
2.  **指示:** 「このドキュメントの内容を要約し、クライアント向けの説明資料（スライド形式）に書き直してください」
3.  **変換:** 出力されたMarkdownをVS Codeで開き、Marp機能で「Export Slide Deck...」→「PowerPoint (.pptx)」を選択。

## 3. 図解の自動化について

Antigravityは **Mermaid記法** の生成に優れています。MarpはMermaidをサポートしているため、テキストだけでフローチャートやガントチャートを描画し、そのままスライドに埋め込めます。

*   **指示例:** 「このプロセスの流れを、Mermaidのフローチャートで可視化してスライドに追加して」

## 4. プロンプト例（スライド作成用）

| 目的 | プロンプト例 |
| :--- | :--- |
| **スライド構成** | 「Marp形式で、以下の内容をスライド10枚にまとめて。各スライドは『タイトル』『箇条書き3点』『補足（スピーカーノート）』で構成して」 |
| **図解作成** | 「現在のシステム構成と、提案する新構成の比較を、Mermaidのクラス図かフローチャートで表現して」 |
| **リバイス** | 「この文章は堅苦しいので、プレゼン用に短く簡潔なフレーズに書き換えて。体言止めを使って」 |

## 5. 環境セットアップ（ユーザー作業）

1.  VS Codeに拡張機能 **"Marp for VS Code"** をインストールしてください。
2.  作成されたテンプレートファイル（`.md`）を開きます。
3.  プレビュー画面を表示（Markdownプレビューボタン）すると、スライドとして表示されます。
4.  コマンドパレット (`Cmd+Shift+P`) から `Marp: Export Slide Deck...` を実行し、`.pptx` を選べば完了です。
