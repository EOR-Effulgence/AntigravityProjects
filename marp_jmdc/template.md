# テーブルの例

| 項目 | 2023年 | 2024年 | 増減 |
|:-----|-------:|-------:|:----:|
| 売上 | 1,000 | 1,200 | ↑ |
| 利益 | 200 | 250 | ↑ |
| 顧客数 | 500 | 480 | ↓ |

<!--
テーブルの記法:
- 1行目: ヘッダー
- 2行目: 配置指定 (:--- 左揃え, :---: 中央, ---: 右)
- 3行目以降: データ行

スタイリング:
- ヘッダーは自動的にグリーン背景
- 偶数行に薄いグレー背景
-->

# 強調ボックス

通常のテキストです。

> **重要なポイント**
> 
> この部分は強調されて表示されます。
> blockquoteを使用すると、ライトグリーンの背景と
> 左側のグリーンバーで目立つボックスになります。

続きのテキストです。

<!--
強調ボックス（blockquote）の使い方:
- > で始める行が強調ボックスになります
- 複数行にわたって使用可能
- 注意事項や重要なポイントの強調に最適
-->

# コードの表示

インラインコード: `const example = "Hello";`

コードブロック:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("JMDC");
```

<!--
コードの記法:
- インライン: バッククォート1つで囲む `code`
- ブロック: バッククォート3つで囲み、言語名を指定
-->
