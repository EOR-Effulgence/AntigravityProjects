# JMDC PowerPointスライド生成 鉄板プロンプト集

このドキュメントは、AIエージェントがJMDCテンプレートを使用して営業スライド・提案書・報告書を生成する際に使用するプロンプト集です。

---

## テンプレート情報

**ファイル**: `JMDC2022_16対9(標準)_v1.1.pptx`  
**サイズ**: 16:9（13.33" x 7.50"）

### 利用可能なレイアウト

| レイアウト名 | 用途 | 主なプレースホルダー |
|-------------|------|---------------------|
| 表紙 | プレゼンの始まり | タイトル、サブタイトル、日付/著者 |
| 目次 | 構成の説明 | タイトル、アジェンダリスト、フッター |
| 中見出し | セクション区切り | サブタイトル、タイトル |
| コンテンツ | 本文・詳細説明 | タイトル、サブタイトル、コンテンツ、フッター |
| 裏表紙 | プレゼンの終わり | （なし） |

---

## JSON出力フォーマット

AIエージェントは以下のJSON形式でスライドデータを出力してください：

```json
{
  "clear_existing": true,
  "slides": [
    {
      "layout": "表紙",
      "title": "タイトル",
      "subtitle": "サブタイトル"
    },
    {
      "layout": "目次",
      "title": "目次",
      "body": "1. はじめに\n2. 課題\n3. ソリューション\n4. まとめ"
    },
    {
      "layout": "中見出し",
      "subtitle": "セクション1",
      "title": "はじめに"
    },
    {
      "layout": "コンテンツ",
      "title": "詳細説明",
      "content": ["ポイント1", "ポイント2", "ポイント3"],
      "footer": "© JMDC"
    },
    {
      "layout": "裏表紙"
    }
  ]
}
```

## 営業スライド用プロンプト

### 1. 商品紹介プロンプト

```
以下の情報をもとに、JMDCテンプレートを使用した商品紹介スライドのJSONを作成してください。

【入力情報】
- 商品名: [商品名]
- ターゲット顧客: [想定顧客]
- 主な特徴: [特徴1, 特徴2, 特徴3]
- 競合との差別化ポイント: [差別化ポイント]
- 価格帯: [価格情報]

【スライド構成】
1. 表紙（商品名、キャッチコピー）
2. 目次
3. 中見出し「課題認識」
4. コンテンツ「ターゲット顧客の課題」
5. 中見出し「ソリューション」
6. コンテンツ「商品の特徴」x2〜3枚
7. コンテンツ「競合比較」
8. コンテンツ「価格・導入ステップ」
9. 裏表紙

JSON形式で出力してください。
```

### 2. 顧客事例プロンプト

```
以下の顧客事例をもとに、営業用の事例紹介スライドのJSONを作成してください。

【入力情報】
- 顧客名: [顧客名]（匿名化が必要な場合は「A社」等）
- 業種: [業種]
- 導入前の課題: [課題]
- 導入したソリューション: [サービス/製品名]
- 導入効果: [定量的効果があればその数値]
- 顧客の声: [コメント]

【スライド構成】
1. 表紙「導入事例：[顧客名]様」
2. コンテンツ「企業概要」
3. コンテンツ「導入前の課題」
4. コンテンツ「ソリューション」
5. コンテンツ「導入効果」
6. コンテンツ「お客様の声」
7. 裏表紙

JSON形式で出力してください。
```

## 使用方法

1. 上記のプロンプトを選択し、【入力情報】部分を実際の情報に置き換える
2. AIエージェントにプロンプトを送信してJSONを生成
3. 生成されたJSONを `input.json` として保存
4. 以下のコマンドでPPTXを生成：
   ```bash
   python scripts/generate_slide.py input.json output.pptx
   ```
