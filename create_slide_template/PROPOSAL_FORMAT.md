# Proposal Markdown Format Specification (`PROPOSAL_FORMAT.md`)

このドキュメントでは、スライド生成のためのMarkdownフォーマットを定義します。
このフォーマットに従って書かれたMarkdownファイル (`proposal.md`) を、`md_to_slide.py` が解析してPowerPointファイルを生成します。

## 1. 基本構造

*   **スライド区切り**: `---` (水平線) でスライドを区切ります。
*   **レイアウト指定**: スライドの先頭に `<!-- layout: レイアウト名 -->` の形式でマジックコメントとして記述します。
*   **スライドタイトル**: `# タイトル` (H1) で記述します。
*   **本文**: 通常のテキストや箇条書きで記述します。

### 例

```markdown
<!-- layout: 表紙 -->
# AIを活用した業務効率化のご提案
## 〜次世代のワークスタイルへ〜
2025年12月28日 株式会社サンプル

---

<!-- layout: 目次 -->
# 本日のアジェンダ

1. 現状の課題
2. ソリューション概要
3. 導入効果
4. 費用感

---

<!-- layout: コンテンツ -->
# 現状の課題

多くの企業で、事務作業に多くの時間が取られています。

*   **データ入力**: 手作業によるミスが発生しやすい
*   **レポート作成**: 複数のシステムからデータを集めるのが大変
```

## 2. チャート (グラフ) の記述

チャートはコードブロックとして記述します。言語指定子に `chart:タイプ` を使用します。
データはCSV形式（カンマ区切り）またはJSON形式で記述可能です。ここでは直感的なCSV(Excelライク)形式を推奨します。

### 構文
````markdown
```chart:種類
タイトル（省略可）
ヘッダー1, ヘッダー2, ヘッダー3
ラベル1, 値1, 値2
ラベル2, 値3, 値4
```
````

### チャートの種類 (`chart:xxx`)
*   `column`: 縦棒グラフ (Clustered Column)
*   `bar`: 横棒グラフ (Clustered Bar)
*   `line`: 折れ線グラフ
*   `pie`: 円グラフ
*   `stacked_column`: 積み上げ縦棒グラフ

### 例: 売上推移（縦棒グラフ）
````markdown
```chart:column
年度別売上推移
年度, 売上(億円), 利益(億円)
2023, 120, 10
2024, 150, 18
2025, 200, 35
```
````

## 3. 表 (Table) の記述

Markdown標準のテーブル記法を使用します。

```markdown
| 機能 | Standard | Pro | Enterprise |
| :--- | :---: | :---: | :---: |
| ユーザー数 | 5 | 20 | 無制限 |
| ストレージ | 10GB | 100GB | 1TB |
| サポート | メール | チャット | 24時間電話 |
```

## 4. 画像の配置

Markdown標準の画像記法を使用します。
配置場所を指定したい場合、画像リンクの直後に配置情報をコメントで追記する拡張も検討しますが、基本は自動配置（右側など）とします。

```markdown
![オフィスの様子](output/images/office.png)
<!-- position: right -->
```

## 5. ノート (Speaker Notes)

スライドには表示されず、スピーカーノートに記載したい内容は `<!-- note: ... -->` で記述します。

```markdown
<!-- note: ここで強調したいのは、コスト削減効果よりも従業員満足度の向上です。 -->
```

## 6. 完全なサンプル

```markdown
<!-- layout: 表紙 -->
# 病院経営DXプロジェクト
## AIによる業務変革
2025年12月28日

---

<!-- layout: コンテンツ -->
# 導入効果の試算

AI導入により、年間で約30%のコスト削減が見込まれます。

```chart:column
コスト削減効果
項目, 導入前, 導入後
人件費, 100, 70
ミス対応, 20, 5
その他, 10, 8
```

<!-- note: ミス対応コストの大幅な削減に注目してください。 -->
```
